/*
Recommended SCSS Structure:
- _sass/00-settings/   // Variables, color tokens, spacing, fonts
- _sass/10-base/       // Reset, typography, base elements
- _sass/20-layout/     // Layout, grid, containers
- _sass/30-components/ // Buttons, cards, nav, forms, etc.
- _sass/40-utilities/  // Utility classes (margin, padding, flex, etc.)

Best Practices:
- Use CSS custom properties for color and spacing tokens in root-vars.css
- Keep all brand colors, fonts, and breakpoints in _settings
- Use BEM or utility-first naming for components
- Modularize: each section/component gets its own partial
- Use @import or @use to compose main style.scss
*/
